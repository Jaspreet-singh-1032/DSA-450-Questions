# Convert a normal BST into a Balanced BST

Difficulty Level : Medium

Given a Binary Search Tree, modify the given BST such that it is balanced and has minimum possible height. Return the balanced BST.

**Examples :**

```
Input:
       30
      /
     20
    /
   10
Output:
     20
   /   \
 10     30
```
