# Find min and max value in a BST

Difficulty Level : Easy

Given the root of a Binary Search Tree. The task is to find the minimum valued element in this given BST.

**Examples :**

```
Input:
           5
         /    \
        4      6
       /        \
      3          7
     /
    1
Output: 1
```
