# Subset Sum Problem

Difficulty Level : Medium

Given an array arr[] of size N, check if it can be partitioned into two parts such that the sum of elements in both parts is the same.

**Examples :**

```
Input: N = 4
arr = {1, 5, 11, 5}
Output: YES
Explanation: 
The two parts are {1, 5, 5} and {11}.
```
