# Print all palindromic partitions of a string

Difficulty Level : Medium

Given a String S, Find all possible Palindromic partitions of the given String.

**Examples :**

```
Input:
S = "geeks"
Output:
g e e k s
g ee k s
Explanation:
All possible palindromic partitions
are printed.
```
