# Binomial Coefficient Problem

Difficulty Level : Medium

Given two integers n and r, find nCr. Since the answer may be very large, calculate the answer modulo 109+7.
Note : If r is greater than n, return 0.

**Examples :**

```
Input: n = 3, r = 2
Output: 3
Explaination: 3C2 = 3. 
```
