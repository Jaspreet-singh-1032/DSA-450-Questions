# Mirror of a tree

Difficulty Level : Easy

Given a Binary Tree, convert it into its mirror.

**Examples :**

```
Input:
      1
    /  \
   2    3
Output: 3 1 2
Explanation: The tree is
  1    (mirror)     1
 /  \    =>        /  \
2    3           3   2
The inorder of mirror is 3 1 2
```
