# Right View of Tree

Difficulty Level : Easy

Given a Binary Tree, find Right view of it. Right view of a Binary Tree is set of nodes visible when tree is viewed from right side. Return the right view as a list. 

**Examples :**

```
Input:
       1
    /    \
   3      2
Output: 1 2
```
