# Level order traversal

Difficulty Level : Easy

Given a root of a binary tree with n nodes, find its level order traversal.
Level order traversal of a tree is breadth-first traversal for the tree.

**Examples :**

```
Input:
    1
  /   \ 
 3     2
Output:
1 3 2
```
