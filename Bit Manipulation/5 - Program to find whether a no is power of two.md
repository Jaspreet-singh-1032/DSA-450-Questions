# Program to find whether a no is power of two

Difficulty Level : Easy

Given a non-negative integer n. The task is to check if it is a power of 2. 

**Examples :**

```
Input: n = 98
Output: false
Explanation: 98 cannot be obtained by any power of 2.
```
