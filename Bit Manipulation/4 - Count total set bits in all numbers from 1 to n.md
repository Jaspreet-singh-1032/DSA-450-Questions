# Count total set bits in all numbers from 1 to n

Difficulty Level : Easy

Given a non-negative integer n. The task is to check if it is a power of 2.

**Examples :**

```
Input: n = 8
Output: true
Explanation: 8 is equal to 2 raised to 3 (2power3 = 8).
```
