# Merge 2 Binary Max Heaps

Difficulty Level : Easy

Given two binary max heaps as arrays, merge the given heaps to form a new max heap.

**Examples :**

```
Input: 
n = 4 m = 3
a[] = {10, 5, 6, 2}, 
b[] = {12, 7, 9}
Output : 
{12, 10, 9, 2, 5, 7, 6}
```
